## RECORDING TIPS

1. **Browser:** Use Chrome in incognito mode (no extensions)
2. **Resolution:** 1920x1080 or 1280x720
3. **Zoom:** Browser at 100% zoom
4. **Speed:** Move mouse deliberately, pause briefly on important elements
5. **Clicks:** Make clicks visible (consider cursor highlighter)
6. **Audio:** Record voiceover separately for cleaner audio, or use a good mic

## KEY POINTS TO EMPHASIZE

- Native MUI component (no third-party dependencies)
- Full accessibility (WCAG 2.1, keyboard, screen readers)
- Touch/swipe gestures
- Multiple transitions
- Theme integration
