# Gatsby example

## How to use

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

```sh
curl https://codeload.github.com/mui/material-ui/tar.gz/v4.x | tar -xz --strip=2  material-ui-4.x/examples/gatsby
cd gatsby
```

Install it and run:

```sh
npm install
npm run develop
```

or:

[![Edit on CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/mui-org/material-ui/tree/v4.x/examples/gatsby)

## The idea behind the example

[Gatsby](https://github.com/gatsbyjs/gatsby) is a static site generator for React.
