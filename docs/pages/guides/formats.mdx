## Format customization

For localization purposes, it may be needed to change display values in the pickers' modals,
because the default formats cannot be idiomatic for some localizations. Utils can help you there.

<Ad />

It's possible to override any of the displayed date values by inheritance of utils passed to
MuiPickersProvider.

#### Patched French pickers

You can use ES6 class syntax or override values with the help of `.prototype` Object property.
