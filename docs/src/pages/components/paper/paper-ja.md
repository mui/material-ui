# Paper

<p class="description">マテリアルデザインでは、紙の物理的特性が画面に変換されます。 </p>

アプリケーションの背景は、1枚の紙の平らで不透明なテクスチャに似ており、アプリケーションの動作は、複数のシートにサイズ変更、シャッフル、およびバインドされる紙の能力を模倣します。

## バリアント

If you need an outlined surface, use the `variant` prop.
