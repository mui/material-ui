# Menubar

<p class="description">A horizontal menu bar that provides quick access to a set of commands, typically used at the top of an application window.</p>

{{"component": "@mui/docs/ComponentLinkHeader"}}

## Basic menubar

A menubar is composed of multiple menus, each with a trigger button and a dropdown containing menu items. This example uses the [Base UI Menubar](https://base-ui.com/react/components/menubar) primitives.
